⚙️ API & Motor Yapılandırması

FenAI, varsayılan olarak dahili veya özel Gemini 2.5 Flash ile entegre çalışır. "Canfenci Akıllı Dağıtım Modu" seçildiğinde görevler girdiğiniz anahtarlara göre otomatik dağıtılır.

🔑 Gemini API Key (Dahili / Özel)

FenAI platformu dışına (örneğin GitHub Pages) kurduğunuzda Gemini modelinin çalışması için buraya kendi ücretsiz Google AI Studio API anahtarınızı girmelisiniz.

🔑 DeepSeek API Key (Özel)

DeepSeek-V3 modelini doğrudan kullanmak için anahtarınız. Bu anahtar girildiğinde Konu Anlatımı, Kavram Testi ve Çalışma Kağıtları bu motorla üretilir.

🔑 OpenRouter API Key (Özel)

Claude 3.5 Sonnet, DeepSeek-R1 ve GPT-4o modellerine OpenRouter üzerinden bağlanmak için kullanılır. (Bağlamlı Soru, Yazılı Sınav ve Sürükle Bırak oyunları için gereklidir).

🗑️ Gizlilik ve Güvenlik

Özel API anahtarlarınız tamamen tarayıcınızda (Local Storage) saklanır, hiçbir harici sunucuya iletilmez.
